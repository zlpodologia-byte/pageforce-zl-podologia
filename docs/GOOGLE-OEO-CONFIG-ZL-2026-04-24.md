Passos:

2. Verificar por arquivo HTML, tag HTML ou DNS.
3. Enviar sitemap.
4. Usar inspeção de URL para solicitar indexação da home.
5. Validar se o Google mostra canonical como `https://www.zlpodologia.com.br/`.

## Google Analytics 4

Pendente técnico:

- Criar propriedade GA4 para ZL Podologia.
- Configurar a variável `NEXT_PUBLIC_GA_ID` na Vercel.
- Fazer novo deploy.
- Validar eventos `wa_click`, `service_select`, `maps_click`, `phone_click`, `quiz_complete`.

Sem `NEXT_PUBLIC_GA_ID`, o site ainda registra eventos de funil via logs da Vercel, mas não envia para GA4.
